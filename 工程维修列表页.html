工程维修列表页

请求类型：POST

API：mobileMerchant_web/repair/getStatusRepairList

请求参数：

参数名 类型 含义 说明
userId String 用户id
statusCode String 类型 100501待分配，100503处理中，100504待确认
page Number 分页页码 默认1页显示20条数据

示例参数：

{
    "version": 1,
    "timestamp": 1552880036039,
    "sign": "YUEWORLD2A72F53B39C",
    "params": {
        "userId": "28685",
        "statusCode": "100501",
        "page": 1
    }
}
