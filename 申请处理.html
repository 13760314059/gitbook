申请处理

请求类型：POST

API：mobileMerchant_web/serviceAll/audit

请求参数：

参数名 类型 含义 说明
userId String 用户id
status String 状态 0 拒绝，1通过
opinion String 备注
serviceId String 服务id

示例参数：

{
    "version": 1,
    "timestamp": 1552895649790,
    "sign": "YUEWORLD2A72F53B39C",
    "params": {
        "userId": "28685",
        "status": "0",
        "opinion": "测试数据，拒绝",
        "serviceId": "3359"
    }
}

示例返回：

{
    "data": "",
    "msg": "success",
    "code": "0"
}
