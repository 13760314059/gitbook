服务记录

请求类型：POST

API：mobileMerchant_web/repair/myRepairApplyList

请求参数：

参数名 类型 含义 说明
userId String 用户id
status String 列表状态 0 进行中，1 已完成
page Number 分页页码

示例参数：

{
    "version": 1,
    "timestamp": 1552892659161,
    "sign": "YUEWORLD2A72F53B39C",
    "params": {
        "userid": "28685",
        "status": "0",
        "page": 1
    }
}
