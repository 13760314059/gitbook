营运审核列表页

请求类型：POST

API：mobileMerchant_web/serviceAll/getAuditedList

请求参数：

参数名 类型 含义 说明
userId String 用户id
status String 状态 0 待审核，1 已审核
page Number 分页页码

示例参数：

{
    "version": 1,
    "timestamp": 1552893703128,
    "sign": "YUEWORLD2A72F53B39C",
    "params": {
        "userId": "28685",
        "status": "0",
        "page": 1
    }
}
