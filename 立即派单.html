请求类型：POST

API：mobileMerchant_web/repair/mechanicList

请求参数：

参数名 类型 含义 说明

serviceId String 服务id

{
    "version": 1,
    "timestamp": 1552887534796,
    "sign": "YUEWORLD2A72F53B39C",
    "params": {
        "userId": "28685",
        "serviceId": 3403
    }
}
